Lesson 1: HTML intro

HTML: HyperText Markup Language - язык разметки гипертекста

Узнать об элементе-ссылке в MDN
  • h1 - heading, заголовок первого порядка, должен быть на странице один
  • img - элемент картинка
  • p - текстовый элемент параграф, отличается размером и стилизацией
  • a - элемент ссылка, аттрибут href - место для url нужного нам адреса
  • ul - unordered list - неупорядоченный список, состоит из li - элементов списка lists
